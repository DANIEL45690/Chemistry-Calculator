## 🎯 Описание

**Chemistry Calculator Pro** — мощный консольный химический калькулятор для студентов, преподавателей и профессионалов. Включает расчёты молярной массы, pH, газовых законов, термодинамики, кинетики и электрохимии.

> Идеален для химиков, студентов и всех, кто работает с химическими формулами.
